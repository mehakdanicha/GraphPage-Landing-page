# HTML & CSS Practice Project

This is a simple web page designed and built using **pure HTML and CSS** for practice purposes. The project is meant to help improve front-end skills, including layout, styling, and responsive design.

## Features

- Clean and semantic HTML structure
- Styled using CSS (no frameworks used)
- Responsive layout for different screen sizes
- Practice with various HTML elements and CSS properties

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Clone the repository:
git clone https://github.com/your-username/your-repo-name.git
Open the project folder.

Open index.html in your browser.
